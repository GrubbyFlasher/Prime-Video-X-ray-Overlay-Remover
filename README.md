# Prime-Video-X-ray-Overlay-Remover


This extension removes X-ray feature and black overlay.<br />
How To Use-<br />
Step 1 - Pin this extension.<br />
Step 2 - Play any Movie/Show.<br />
Step 3 - Click on extension icon.<br />
